# Student Performance Analysis

A statistical analysis project investigating the relationship between student performance, gender, and parental education level using Python.

## Project Overview

This repository contains an exploratory and inferential statistical analysis of a student performance dataset. The objective is to determine whether academic achievement differs by gender and parental level of education.

The analysis includes:

* Data loading and preprocessing
* Exploratory Data Analysis (EDA)
* Descriptive statistics
* Data visualization
* Hypothesis testing

  * Welch's t-test
  * One-way ANOVA
  * Tukey's HSD post-hoc analysis

The work is documented in a Jupyter Notebook and follows a structured analytical workflow commonly used in data science projects.

## Technologies

* Python
* Jupyter Notebook
* pandas
* NumPy
* SciPy
* statsmodels
* Matplotlib
* Seaborn

## Statistical Methods

The project applies several standard statistical techniques:

* Descriptive statistics
* Boxplots and bar plots
* Levene's test for homogeneity of variances
* Welch's independent two-sample t-test
* One-way Analysis of Variance (ANOVA)
* Tukey's Honest Significant Difference (HSD) test

## Running the Project

1. Clone the repository.
2. Create and activate a virtual environment.
3. Install the required dependencies:

```bash
pip install -r requirements.txt
```

4. Launch Jupyter Notebook:

```bash
jupyter notebook
```

5. Open the analysis notebook and execute the cells in order.
